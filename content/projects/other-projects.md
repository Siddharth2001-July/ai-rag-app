# Other Projects (reference list)

Smaller experiments and learning projects on Siddharth's GitHub. Featured here for reference — they're not the headline portfolio pieces, but they show the breadth of things he's tried.

## Gmail Compose Template Extension

A Chrome extension built with the **InboxSDK** that injects a customizable template into Gmail's compose view, so you can pre-populate emails with a default body. Uses the InboxSDK's `Compose.registerComposeViewHandler` to hook into Gmail's UI.

- **Stack:** Vanilla JS, Chrome Extension APIs, InboxSDK
- **Repo:** https://github.com/Siddharth2001-July/gmail-chrome-extension

## Discord Bot

A small Discord bot scaffolded in **TypeScript** using `discord.js` and `dotenv` for config. Built as an experiment with the Discord API.

- **Stack:** TypeScript, discord.js, dotenv
- **Repo:** https://github.com/Siddharth2001-July/discord-bot

## ExcelReader

An Angular 14 app for reading and working with Excel files in the browser, with **Firebase** integration for storage / auth. One of Siddharth's earlier Angular projects.

- **Stack:** Angular 14, Firebase, Bootstrap
- **Repo:** https://github.com/Siddharth2001-July/excelReader

## Blog Website

A classic CRUD blog built with **Express + EJS** (server-rendered, no client framework). Built early on as a way to learn server-side templating, routing, and basic web architecture without a framework abstracting it away.

- **Stack:** Node.js, Express, EJS, Lodash
- **Repo:** https://github.com/Siddharth2001-July/blog-website

## Weather API App

A simple weather-fetching app using a public weather API and Express on the backend. Early Node.js / API integration practice.

- **Stack:** Node.js, Express
- **Repo:** https://github.com/Siddharth2001-July/WeatherApi
